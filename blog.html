Question-2: What is the difference between innerHTML, innerText, and textContent ?

-------------------------------------------------------------------------------------------

1:innerHtml => When you need to access or modify the HTML structure(tags,attributes,and content) 2:innerText Use case =>When you need the visible text content,and it respects the user's styling(e.g.,hidden text wont't be included). 3:textContent => When you need the raw text content and wants the best performance,regardless of CSS styling or hidden content

Question-3: What is event delegation in the DOM?

-------------------------------------------------------------------------------------------

Event delegation is a technique in jaaScript where a single event listener is added to a parent element instead of multiple listeners on individual child element.The event bubbles up from the child to the parent,Where it is handled efficiently

Question-4: What is event bubbling in the DOM?

------------------------------------------------------------------------------------------

The event bubbles up from the child to the parent,Where it is handled efficiently.Where events triggered on child elements travel up the DOM tree to their parent elements.
